What is font-tidop-gis?

font-tidop-gis is a set of icons and graphics I've been using in my projects.
font-tidop-gis icons and font theme is designed mainly for GIS applications and web mapping tools. Feel free to use it in your commercial or open source projects, or really almost whatever you want. font-tidop-gis is free, open source, and GPL friendly!

How to use it?

You can use font-tidop-gis as a font or as SVG symbols or images.

Use an inline element with a class prefixed with ftg- to add a new icon. 

<i class="ftg-poi"></i>

Using a <span> is more semantically correct but a little bit verbose: 

<span class="ftg-polyline-pt"></span>

Or use it as an svg sprite (svg sprites are inlocated in the ./dist/font-tidop-gis.svg file): 

<svg class="font-tidop-gis ftg-2x">
  <use xlink:href="path/to/dist/font-tidop-gis.svg#ftg-polyline-pt" />
</svg>

NPM package

font-tidop-gis is avaliable on NPM. To use font-tidop-gis in your project, install the font-tidop-gis package: 

npm install --save font-tidop-gis

The font is located in the ./font directory, the css and svg sprites are located in the ./css/font-tidop-gis.css and ./dist/font-tidop-gis.svg files of the font-tidop-gis project.

Icon size and color

To increase icon sizes relative to their container, use the ftg-lg (33% increase), ftg-2x, ftg-3x, ftg-4x, or ftg-5x classes.

Fixed width icons

Use ftg-fw class to set icons at a fixed width. Especially designed to use in nav lists & list groups to preserve alignment.

  • Road map

Rotated and flipped icons

Use ftg-rotate* classes to rotate icons (-20, -10, 10, 20, 45, 90, 135, 180, 225, 270 or 315).

Use ftg-flipv or ftg-fliph classes to flip icons vertically or horizontally.

Stacked icons

To stack multiple icons, use the ftg-stack class on the parent. 

<span class="ftg-stack ftg-3x">
  <i class="ftg-compass" style="color:#666;"></i>
  <i class="ftg-compass-needle ftg-rotate45" style="color:#800;"></i>
</span>
